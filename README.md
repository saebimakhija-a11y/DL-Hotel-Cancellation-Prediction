# DL-Hotel-Cancellation-Prediction
A deep learning project focused on predicting hotel booking cancellations using customer and reservation data.
This project was developed as part of a university deep learning final assignment and demonstrates the application of neural networks to solve a real-world business problem in the hospitality industry.

Business Problem:
Hotel booking cancellations create major financial and operational challenges for hotels. 
This model can help with improving operational efficiency and reducing financial loss. Also to improve customer satisfaction by anticipating their behaviour more accurately. 

Dataset:
It has information such as Lead time, customer type, deposit type, meal plan, previous cancellation, among other info. 

Target variable:

'is_canceled'

The project follows a complete machine learning workflow:

Data preprocessing

Missing value handling

Feature scaling

One-hot encoding

Train-validation split

Neural network training

Model evaluation
